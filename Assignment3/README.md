# Assignment 3 (UNO GraphQL + Vue)
## Structure
- `domain/` – UNO domain model.
- `server/` – GraphQL server using the domain model.
- `client/` – Vue + Pinia UI (styled in black/white/gray like Assignment 5).

## Getting started
```bash
npm install               # installs all workspaces
npm run dev -w server     # start the GraphQL server
npm run dev -w client     # start the Vite dev server
```
